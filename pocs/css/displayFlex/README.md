### 3. Code tricks
* Define the number (and the widths) of columns in a grid layout
    ```css
    grid-template-columns: 1fr 1fr 1fr 1fr 1fr 1fr;

    /* shorter equivalent code */
    grid-template-columns: repeat(6, 1fr);
    ```
